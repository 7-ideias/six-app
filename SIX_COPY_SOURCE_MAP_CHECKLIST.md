## 🎯 Padrão de Nomenclatura de Chaves

Todas as chaves seguem o padrão:

```
{secao}.{tipo}.{variacao}?
```

**Exemplos:**
- `hero.title` (chave simples)
- `hero.title.curto` (com variação de comprimento)
- `featureTitles` (array)
- `featureTitles[0]` (elemento específico do array)

**Seções válidas:**
- `nav` — Navegação
- `hero` — Hero section
- `features` — Features
- `demo` — Demonstração
- `pricing` — Preços
- `checkout` — Checkout
- `onboarding` — Onboarding
- `footer` — Rodapé
- `common` — Shared/common
